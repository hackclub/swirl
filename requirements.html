Requirements


What do I make?


Swirl is the next step after the completion of Boba Drops!
You are allowed to expand on your Boba Drops project due to this!


Your website must...


  • Be original (No copy pasting from ANYWHERE)
  • Have 3 HTML pages with content
  • Have a Favicon & Title
  • Meet all requirements from Boba Drops (on the submission website!)
  • You’re a high schooler
  • This website was made recently
  • Your website is static (uses HTML + CSS)
  • Your website has text, color and a image
  • You haven't gotten a Swirl grant before
  • Have something fun & unique!
  • Have at least one scoop…


    • Other ground rules around submissions:

  • No Ai is permitted in any way (includes content).
  • JavaScript is allowed so long as it is not the focal point of the website. (ex. games website)
  • If your website was submitted for Boba Drops, it has been expanded on since.
  • Frameworks and other languages are not permitted.

    • Scoops!

    You can't eat ice cream without any scoops!

    Every 3 unique features that are added, you get a scoop!
    Prizes increase with more scoops for a maximum of 3!
    Every submission needs at least one scoop!
    Features that are required do not count for a scoop



    Here's a list for inspiration:


  • Have a footer
  • 404 Page
  • Link to another website (could be the Github repo!)
  • Special link (email, phone, etc)
  • Special text Formatting
  • HTML list
  • Bookmark (only if page is long enough)
  • 3 Paragraphs on a page, each with unique CSS
  • Custom imported Font (Check out Google Fonts)
  • HTML Button
  • CSS :hover or other pseudo-class
  • HTML Audio
    • Features not listed here are welcome, as long as their technical and cool! JavaScript code does not count towards Scoops. Ask in #swirl for any clarification!
